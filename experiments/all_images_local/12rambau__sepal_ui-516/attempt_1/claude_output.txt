## Summary

Successfully deprecated `GADM_ZIP_DIR` in the AOI model. The constant was no longer being used (the codebase now uses `tempfile.TemporaryDirectory()`), so it has been converted to a property that emits a deprecation warning.

### Changes Made:

**1. sepal_ui/aoi/aoi_model.py:**
- Added `import warnings` to imports
- Converted `GADM_ZIP_DIR` from a class constant to a `@property` method
- Added deprecation warning with `DeprecationWarning`
- Added docstring with deprecation notice (version 2.3.2)
- Removed `GADM_ZIP_DIR.mkdir(parents=True, exist_ok=True)` call

**2. docs/source/modules/sepal_ui.aoi.AoiModel.rst:**
- Removed `~AoiModel.GADM_ZIP_DIR` from the attributes list

The deprecated property will now emit a warning when accessed, alerting users that the code has been removed and will be removed in a future version.
